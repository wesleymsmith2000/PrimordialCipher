How To Use This Folder

  • Open docs-center/index.html directly in a browser for local reading.
  • Keep code-oriented material under docs-center/code-documentation/.
  • Keep player-facing help under docs-center/gameplay-documentation/.
  • Mermaid diagrams render when the browser can load Mermaid from its CDN.

This is a static first pass. It can later be migrated to Sphinx, VitePress, Docusaurus, or generated TypeDoc while preserving the same two-section information architecture.
